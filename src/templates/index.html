WSC API

API URL. A response code of 0 indicates a succesful connection. A response code of -1 indicates an error. Both are delivered through the response body.

            
                GET
                34.42.125.145:8080/api/home
                responce = {
                    "code" : 0,
                    "body" : jsonObject
                }

Register new user.

            
                POST
                34.42.125.145:8080/api/register
                json = {
                    "service" : "service1",
                    "username" : "yourchoice",
                    "password" : "yourchoice"
                }

Login as a registered user.

            
                POST
                34.42.125.145:8080/api/login
                json = {
                    "service" : "service1",
                    "username" : "yourchoice",
                    "password" : "yourchoice"
                }

Change username.

            
                POST
                34.42.125.145:8080/api/update/username
                json = {
                    "username" : "existingusername",
                    "password" : "yourpassword",
                    "new_username" : "yourchoice"
                }

Change password.

            
                POST
                34.42.125.145:8080/api/update/password
                json = {
                    "username" : "existingusername",
                    "password" : "yourpassword",
                    "new_password" : "yourchoice"
                }

Delete account.

            
                POST
                34.42.125.145:8080/api/delete
                json = {
                    "username" : "existingusername",
                    "password" : "yourpassword",
                }

Logout.

            
                GET
                34.42.125.145:8080/api/logout
